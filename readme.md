# Laravel Sentinel Boilerplate 

## How To Install :
`
git clone https://github.com/n1arash/sentinel-boilerplate.git`
`composer install`
`php artisan key:generate`
`php artisan migrate`
`setup .env file`
`php artisan serve`


## Help !:
Use PostMan For Testing The API.
