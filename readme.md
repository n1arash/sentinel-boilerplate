# Laravel Sentinel Boilerplate 

## How To Install :
`
git clone https://github.com/n1arash/sentinel-boilerplate.git`<br> 
`composer install`<br> 
`php artisan key:generate`<br> 
`php artisan migrate`<br> 
`setup .env file`<br> 
`php artisan serve`<br> 


## Help !:
Use PostMan For Testing The API.
